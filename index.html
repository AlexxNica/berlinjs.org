Berlin.JS is a usergroup focused on JavaScript and related topics. We meet regularly on the 3rd Thursday each month at 7p.m. at co.up Offices, Adalbertstraße 7-8 in Berlin-Kreuzberg.

Next meetup on March 20th

Functional Interface Programming

Martin Klepsch

Over the last few weeks I explored a more functional approach to programming interfaces mostly inspired by ideas behind Facebook's React.js. One of those ideas is immutability which also happens to be a core feature in Clojurescript, a LISP targeting the browser.
Because of that match in concepts the Clojurescript community has developed some interesting projects based on React.js, that I think are really promising for building modern web applications with complex interfaces.

I want to talk about the ideas behind those projects and how they evolved from the ideas already present in React.js.
Topics: React.js, Clojurescript, Functional Programming, DOM Manipulation

No Framework

Bodo Kaiser

Can't see anymore jQuery or other called "full stack super" framework anymore? Then I want to show you how you can write readable, logic heavy code for the browser without being forced to any single framework by setting up a simple web app project in front of the audience.

Generally I want to share the experience I made coding an online store in order to to free the people up from browser frameworks :)

Building the Node.js for iOS

Sam Rijs

In 2014, writing iOS apps in Javascript is not a new idea. And yet, ascending from the unpleasant pile of the most diverse and proprietary APIs is something wonderful: Warm and fuzzy — somehow familiar ;)

While providing you with an overview on the various ways and pitfalls of using Javascript on Apple's mobile platform, Sam is also going to present his current project: A node.js implementation for iOS.

Code of Conduct

Long story short: Be awesome to each other.
