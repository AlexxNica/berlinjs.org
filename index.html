Berlin.JS is a usergroup focused on Javascript and related topics. We meet regularly on the 3rd Thursday each month at 7p.m. at co.up Offices, Adalbertstraße 7-8 in Berlin-Kreuzberg.

Mark your calendars: Berlin.JS is back

Next meetup on November 17th

Making mobile UIs fast when you're always an edge case

Horia Dragomir
@hdragomir

Horia will show you the hoops you have to jump through to have a sleek, fast and responsive User Interface for handhelds. Most of his learnings come from building one of the first mobile HTML5 Games for Facebook Mobile, Magic Land Mobile.

Implementing linewrapping into an editor

Marijn Haverbeke
@marijnjh

Marijn presented CodeMirror at BerlinJS a few months ago. This time, he'll describe the process of writing some extensions to the editor, most notably how it supports text wrapping and code folding on huge documents without compromising on responsiveness.

Partial function application with JavaScript

Adrian Lang
@adrianlang

Adrian will introduce you to partial function application and present a toolkit allowing JavaScript functions to be partially applied. Afterwards, he would like to talk about use cases for partial function application and discuss whether you actually would want to use it.

Are We Playing Yet?

Yves Van Goethem
@yvg
Tomás Senart
@tsenart

Last week SoundCloud released Are We Playing Yet?, a HTML5 audio test suite. In their lightning talk Yves and Tomás will go trough it's architecture, it's goals, what the state of it is right now and how you can participate to make Audio rock!

Tracking Frontend errors for mobile sites

Yves Van Goethem
@yvg

In his second lightning talk Yyes will share with us how SoundCloud tracks frontend errors on their new mobile site, what solutions they came up with and how they use existing tools for that task.
