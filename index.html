Create.js and inline web editing

Henri Bergius
Nemein

Create.js is a comprehensive open source web editing interface. It is designed to provide a fully browser-based WYSIWYG environment for managing content.

Thanks to Backbone.js and VIE, Create can be adapted to work on almost any server backend.

Node.js Quadcopter Programming

Felix Geisendörfer

Interested in node.js and excited by autonomous robots?

Come and find out about programming Parrot's AR Drone quadcopters to perform autonomous tasks using their built-in sensors, network protocol and opencv powered image recognition.
